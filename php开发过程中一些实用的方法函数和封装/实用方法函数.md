# PHP 实操实用方法函数

## 1. 二维数组按照键值排序
```
/**
 * 二维数组按照键值排序
 *
 * @param array  $arr 待排序数组
 * @param string $key 键值
 * @param string 
 * @return mixed
 */
function sortByKey($arr, $key, $is_asc = true) 
{
  
  if ($is_asc) {
    $sort_type = SORT_ASC;
  } else {
    $sort_type = SORT_DESC;
  }
  
  array_multisort(array_column($arr, $key), $sort_type, $arr);
  
  return $arr;
  
}
```

## 2. 根据二维数组中的部分键值判断二维数组中是否有重复值,并返回重复值
```
/**
 * 作用：根据二维数组中的部分键值判断二维数组中是否有重复值,返回重复值
 *
 * @param array $arr 目标数组
 * @param array $keys 要进行判断的键值组合的数组
 *
 * @return array 重复的值
 */
function getArrayRepeat($arr = [], $keys = [])
{
  $unique_arr = [];
  $repeat_arr = [];
  foreach ($arr as $k => $v) {
    $str = "";
    foreach ($keys as $a => $b) {
      $str .= "{$v[$b]},";
    }
    if (!in_array($str, $unique_arr)) {
      $unique_arr[] = $str;
    } else {
      $repeat_arr[] = $v;
    }
  }
  return $repeat_arr;
}
```
